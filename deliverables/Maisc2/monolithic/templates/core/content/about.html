> MAISC2 Project
Method to Support Semantic Interoperability in Command and Control or, in Portuguese, "Método para Apoiar a Interoperabilidade Semântica em Comando e Controle" (MAISC2) is an academic research project that uses Natural Language Processing (NLP) to process messages exchanged in a command and control environment to help the receiver better understand the message.
