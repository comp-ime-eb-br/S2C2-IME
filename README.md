# Sistemas de Sistemas de Comando e Controle - S2C2 -IME

Esta é a página inicial relativa aos artefatos desenvolvidos na modelagem conceitual dos elementos constituintes do projeto de Sistemas de Sistemas de Comando e Controle a cargo do Instituto Militar de Engenharia - IME. O site do projeto para navegação pelos elementos citados está disponível no link https://comp-ime-eb-br.github.io/S2C2-IME/.

## Organização dos Artefatos

1. *diagrams* - diagramas da modelagem conceitual construídos no Visual Paradigm - https://github.com/comp-ime-eb-br/S2C2-IME/tree/main/diagrams
2. *docs* - documentos pertinentes ao projeto e artigos publicados
3. *onto* - ontologias operacionais desenvolvidas para representar os elementos dos Sistemas de Sistemas de Comando e Controle 
4. *deliverables* - códigos-fonte dos protótipos, scripts de emulação e teste de funcionalidade, e provas de conceito desenvolvidos
5. *onto* - ontologias operacionais e suas respectivas documentações
6. *publi* - artigos, teses, dissertações, projetos finais e relatórios do projeto.
