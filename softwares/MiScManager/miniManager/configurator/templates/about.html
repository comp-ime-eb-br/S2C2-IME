MiScManager é um gerenciador/configurador para simulações de comunicações militares que utilizam o Mininet-WiFi como emulador de redes, focadas em cenários militares criados a partir de uma ontologia de referência. Através do MiScManager, os usuários podem configurar tanto o cenário militar quanto as especificações da rede de comunicação sem fio, seja importando a configuração de uma ontologia ou manualmente.




MiScManager é uma extensão do MiniManager. Clique no botão "Baixar MiniManager Paper" para baixar o artigo sobre o MiniManager.




Clique no botão de download abaixo para baixar a ontologia base para a criação de cenários militares.
